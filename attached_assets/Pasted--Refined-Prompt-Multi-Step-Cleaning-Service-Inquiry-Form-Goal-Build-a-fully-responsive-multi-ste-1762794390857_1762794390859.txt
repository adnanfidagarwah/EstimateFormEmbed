✅ Refined Prompt: Multi-Step Cleaning Service Inquiry Form

Goal:
Build a fully responsive multi-step residential cleaning service inquiry form with modern UI/UX. The form should collect detailed information across multiple steps and submit all data to a server-side script that sends the results via SMTP email.

✅ 1. General Requirements
Design & UI

Clean, modern theme (purple/white/minimal/minimal shadow).

Header text:

“We create tailored cleaning solutions for you.”

“The industry’s most trusted cleaning expert”

“Live Agents — Free Estimates”

Progress bar or step indicator across all steps.

Validation on each step before moving forward.

Smooth transitions between steps.

Tech Stack

HTML + CSS + JavaScript (front-end)

PHP / Node / Python (any server language)

SMTP for final email delivery

Google Places API for address autocomplete

Send to: info@clientdomain.com (or provided email)
